# M5Stack-Core2_examples 
★日本語の説明は下に

A small program example using M5Stack Core2

This repository contains a collection of **minimal examples for M5Stack Core2**,  
designed to be simple, clean, and easy to understand.
Each example is a standalone code snippet.

## ▶ How to Use
1. Create a PlatformIO project for M5Stack Core2
2. Copy the example code into `src/main.cpp`
3. Build & upload

＊Add libraries as needed.

## 📚 Example List

- abc_buttons.cpp
  
  Soft-colored button indicator using A/B/C touch buttons.

# 🇯🇵 日本語版 README

M5Stack Core2 を使用した簡単なプログラム例です。
各cppファイルは独立したコード例になっています。

## ▶ 使用方法
1. VSCodeのPlatformIOで「M5Stack Core2用」のプロジェクトを作成します。
2. 各サンプルコードの内容を `src/main.cpp` にコピーします。
3. ビルドしてM5Stack core2にアップロードします。（焼く）
※必要に応じてライブラリを追加してください。
